# Kinza Ghanchi Type & Interaction Semester 2
## Functions 
### This is a project for my Type & Interaction class at Parsons, which focuses on HTML, CSS and Javascript. This project focuses on a problem⁠ I have identified in my own life, and the lives of others - time. More specifically, how some of us are unable to keep up with time, cannot track time, and mismanage time. Thus, using the tools, technologies, and techniques I have learnt in this course, I have designed and implemented my own solution: A dynamic to-do list that also serves as a time-tracker.
